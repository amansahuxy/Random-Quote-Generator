# 📝 Random Quote Generator

A simple Random Quote Generator built using HTML and JavaScript. The application fetches a random English quote from an API, translates it into Hindi, and provides options to copy, like, and share the quote.

## 🚀 Features

* Generate random quotes
* Translate quotes from English to Hindi
* Copy quotes to the clipboard
* Like or unlike a quote
* Share quotes on WhatsApp
* Loading indicator while fetching data
* Error handling for failed API requests

## 🛠️ Technologies Used

* HTML5
* JavaScript
* Fetch API

## ▶️ How to Run

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Click **Generate Quote** to fetch a new quote.

## 🧠 How It Works

1. Click the **Generate Quote** button.
2. A random English quote is fetched from the API.
3. The quote is translated into Hindi.
4. Both quotes are displayed on the page.
5. You can:

   * Copy both quotes to the clipboard.
   * Like or unlike the quote.
   * Open WhatsApp to share the quote.

## 🌐 APIs Used

* **DummyJSON Quotes API** – Fetches random English quotes.
* **MyMemory Translation API** – Translates English quotes into Hindi.
